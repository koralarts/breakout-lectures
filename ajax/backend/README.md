# Ajax Backend

## Instructions

1. Run `npm i`
2. Run `npm start`
3. Access the API through: `http://localhost:8888`

## Endpoints

`GET /reset` - Reset the in-memory database

`GET /items` - Get all items

`GET /items/:id` - Get a specific item based on ID

`POST /items` - Create a new item

`PUT /items/:id` - Update item based on ID

`DELETE /items/:id` - Delete item based on ID
